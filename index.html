HunnyPotr by Recce7

Technical Documentation For The Easily Extensible Honey Pot

Purpose of Project

Create a one-click-install honey pot that any Security Engineer or Systems Administrator can easily write custom plugins for applications or services running on their network. The honey pot needs to display critical data to the user in an easy to read and navigate format. This tool is useful to show what kind of attacks are going on and verify that the actual services are guarded appropriately.
